# SQL-LEARNING 📚

Hands-on SQL practice scripts built with **PostgreSQL**.

This repository is a structured learning space that starts with schema creation and moves through querying, data manipulation, joins, aggregation, and common database functions.

## Purpose

- Practice SQL fundamentals with real examples
- Keep learning scripts organized for quick revision
- Track progress while learning SQL step by step

## Repository Files

| File | What it covers |
| --- | --- |
| `1_DATASET_CREATION.sql` | Create tables (`Categories`, `Products`, `Customer`, `Orders`), define relationships, and insert sample data |
| `1_DATASET_QUERYING.sql` | Basic `SELECT` queries, filtering (`WHERE`, `AND`, `OR`), and sorting (`ORDER BY`) |
| `2_DATA_MANIPULATION.sql` | `INSERT`, `UPDATE`, `DELETE`, `TRUNCATE`, and `DROP` examples |
| `3_ADVANCED_DATA_QUERYING.sql` | `JOIN`s, `UNION`/`UNION ALL`, `CASE`, `GROUP BY`, and `HAVING` |
| `3_DATA_AGGREGATION_FUNCTIONS.sql` | `COUNT`, `SUM`, `MAX`, `MIN`, `AVG`, and `STDDEV` |
| `4_DATABASE_FUNCTIONS.sql` | Math and date functions such as `ABS`, `ROUND`, `SQRT`, `CURRENT_DATE`, `EXTRACT`, and `DATE_PART` |

## Topics Covered

- Table creation with primary keys and foreign keys
- Querying and filtering data
- Modifying and removing data
- Joining tables and combining result sets
- Aggregation and grouped analysis
- Built-in PostgreSQL mathematical and date functions

## Getting Started

1. Install [PostgreSQL](https://www.postgresql.org/download/).
2. Create a database (for example, `sql_learning`).
3. Run the SQL files in this order:
   1. `1_DATASET_CREATION.sql`
   2. `1_DATASET_QUERYING.sql`
   3. `2_DATA_MANIPULATION.sql`
   4. `3_ADVANCED_DATA_QUERYING.sql`
   5. `3_DATA_AGGREGATION_FUNCTIONS.sql`
   6. `4_DATABASE_FUNCTIONS.sql`

## Notes

- Scripts are named to reflect learning flow and recommended order.
- SQL comments inside files explain each section.
- The repository may continue to expand with new SQL topics.
